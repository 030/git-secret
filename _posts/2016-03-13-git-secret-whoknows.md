git-secret-whoknows - prints email-labels for each key in the keyring.
======================================================================

## SYNOPSIS

    git secret whoknows


## DESCRIPTION
`git-secret-whokowns` prints list of email addresses which are used as labels for currently public keys added to the local keyring.


## OPTIONS

    -h  - shows this help.
